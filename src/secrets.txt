FILE: secrets.h  (no .cpp counterpart)
================================================================================

PURPOSE
-------
Stores your home WiFi credentials in one place. Edit this file before
flashing — nothing else needs to change to connect to a different network.


CONTENTS
--------
  WIFI_SSID   The name (SSID) of your home WiFi network.
  WIFI_PASS   The WPA2 password for that network.


HOW TO EDIT
-----------
Open src/secrets.h and replace the placeholder values:

  #define WIFI_SSID  "YourNetworkName"
  #define WIFI_PASS  "YourPassword"

Then rebuild and flash. The device will connect to this network whenever
WiFi upload mode is entered (hold SET at boot, or Settings → WiFi Upload).


SECURITY
--------
This file contains plaintext credentials.

  * Do NOT commit secrets.h to a public git repository.
  * Add it to .gitignore if you version-control this project:
      echo "src/secrets.h" >> .gitignore

  The file is intentionally kept out of source control.
  Share it only through secure, private channels (e.g. password manager).


USED BY
-------
  network.cpp — passed directly to WiFi.begin(WIFI_SSID, WIFI_PASS).
